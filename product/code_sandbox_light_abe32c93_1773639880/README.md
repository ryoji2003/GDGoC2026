# SpeakUp - コミュニケーション能力向上アプリ

AIとのリアルな会話練習で、対人コミュニケーション能力を向上させるモックアプリケーションです。

![SpeakUp Logo](https://img.shields.io/badge/SpeakUp-v1.0-6C63FF?style=for-the-badge)
![License](https://img.shields.io/badge/license-MIT-success?style=flat-square)

## 🎯 プロジェクト概要

**対象者**: コミュニケーション能力に不安がある方、対人関係に困っている方

**コンセプト**: 人間そっくりなAIとシチュエーション別の会話練習を行い、練習後にAIから詳細な評価と改善点のフィードバックを受けられるアプリです。

## 🚀 使い方

### 基本的な使用方法

1. **index.html を開く**
   ```
   ブラウザで index.html を開くだけで動作します
   ```

2. **ホーム画面から練習を開始**
   - 「今日の練習を始める」ボタンをクリック
   - シナリオ選択画面へ遷移

3. **シナリオを選択**
   - カテゴリフィルターで絞り込み可能
   - 好きなシナリオカードをクリック

4. **会話練習**
   - AIが話しかけてくる
   - マイクボタンをタップして返答（モック動作）
   - 会話が自動的に進行
   - 「終了する」ボタンで終了可能

5. **評価を確認**
   - 総合スコアと詳細評価を確認
   - フィードバックを読む
   - 次の練習へ進むまたは再挑戦

6. **学習記録を確認**
   - 底部ナビの「記録」をタップ
   - グラフで成長を可視化
   - 獲得したバッジを確認

### インタラクション一覧

| 画面 | 機能 | 動作 |
|------|------|------|
| ホーム | 今日の練習を始める | シナリオ選択画面へ遷移 |
| ホーム | 最近の練習カード | クリック不可（表示のみ） |
| シナリオ | カテゴリフィルター | シナリオをカテゴリ別に絞り込み |
| シナリオ | シナリオカード | 会話練習画面へ遷移（選択したシナリオ） |
| 会話練習 | マイクボタン | 録音開始・終了（モック動作、2秒後に自動終了） |
| 会話練習 | 終了ボタン | 評価画面へ遷移 |
| 評価 | もう一度練習 | シナリオ選択画面へ戻る |
| 評価 | 次のシナリオへ | シナリオ選択画面へ戻る |
| 評価 | 結果をシェア | Web Share API（モバイル対応） |
| 記録 | 週間・月間切り替え | グラフデータを更新 |
| 全画面 | 底部ナビ | 各画面へ遷移 |
| 全画面 | ダークモード切り替え | ライト/ダークテーマ切り替え |

## 🎯 現在のモック動作

以下の機能はモックアップであり、実際のバックエンド処理は行っていません：

1. **会話練習の自動進行**
   - AIメッセージは事前定義された配列から順次表示
   - タイマーベースで3〜5秒ごとに次のメッセージへ遷移

2. **マイク録音**
   - ボタンをクリックすると録音中アニメーション
   - 2秒後に自動終了し、事前定義された返答テキストを表示

3. **評価スコア**
   - 固定値（72点、各項目60〜85点）を表示
   - 実際の音声認識や評価ロジックは未実装

4. **学習記録データ**
   - グラフは固定値のモックデータを表示
   - 実際のデータ保存・読み込み機能は未実装

5. **プロフィール画面**
   - クリック時にアラート表示（「開発中です」）
   - 画面自体は未実装
